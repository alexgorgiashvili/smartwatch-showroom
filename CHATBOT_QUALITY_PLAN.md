### ცვლილება 1.1 — ქართული მავნე keyword-ების დამატება
`InputGuardService.php`-ში შექმენი ახალი მეთოდი `isHarmfulGeorgian()`:

```php
private function isHarmfulGeorgian(string $message): bool
{
    $normalized = mb_strtolower($message);

    $harmfulPatterns = [
        // PII / data theft
        '/(მოვიპარო|მოპარვა|მოიპარე|გამოტყუება)/u',
        '/(ბარათის\s+მონაცემ|საკრედიტო\s+ბარათ|პინ\s*კოდ)/u',
        '/(პაროლ|password|სეკრეტულ|კონფიდენციალ)/u',
        '/(მომხმარებლ\w+\s+მონაცემ|პერსონალურ\s+ინფორმაცი)/u',
        '/(შეკვეთ\w+\s+დეტალ|სხვა\s+მომხმარებელ)/u',
        // DB / System attacks
        '/(წაშალე?\s+database|drop\s+table|წაშალე?\s+ბაზა)/u',
        '/(ჰაკი|ჰაკერ|გატეხვა|გატეხე|შეღწევა)/u',
        '/(admin\s*panel|ადმინ\s*პანელ)/u',
        // Violence / illegal
        '/(აფეთქება|იარაღ|მოკვლა|თავს\s+დაესხ)/u',
    ];

    foreach ($harmfulPatterns as $pattern) {
        if (preg_match($pattern, $normalized) === 1) {
            return true;
        }
    }

    return false;
}
```

## Task 9 — P2: PII Redaction False Positive

### პრობლემა
**sku-005:** "010-02785-01 მოდელი რას მოიცავს?" → `[REDACTED_PHONE]` — SKU კოდი ტელეფონის ნომრად მიიჩნია.

### ფაილი
- `app/Services/Chatbot/InputGuardService.php` — ხაზი 115

### ცვლილება
ტელეფონის ნომრის regex-ს დაამატე negative lookbehind/lookahead პროდუქტ-კოდების ფორმატისთვის:

ახლანდელი:
```php
$redacted = preg_replace('/\+?\d[\d\s\-\(\)]{7,}\d/u', '[REDACTED_PHONE]', $redacted) ?? $redacted;
```

შეცვალე:
```php
// SKU/model codes (e.g., 010-02785-01) usually have dashes between groups,
// phone numbers typically start with + or have 9+ digits without letter context
$redacted = preg_replace(
    '/(?<![A-Za-z\-])\+\d[\d\s\-\(\)]{7,}\d(?![A-Za-z\-])/u',
    '[REDACTED_PHONE]',
    $redacted
) ?? $redacted;
```

**ალტერნატივა (უფრო conservative):** მხოლოდ `+` ნიშნიანი ნომრების redact-ი:
```php
$redacted = preg_replace('/\+\d[\d\s\-]{7,}\d/u', '[REDACTED_PHONE]', $redacted) ?? $redacted;
```

**ტესტი:** `010-02785-01` არ უნდა დაირეაქტოს, `+995555123456` უნდა.
